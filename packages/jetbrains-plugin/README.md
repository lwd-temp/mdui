# mdui

为 mdui 框架提供自动完成、悬停提示等功能的 IntelliJ IDE 插件。

支持以下基于 IntelliJ 的 IDE：

* IntelliJ IDEA
* PyCharm
* WebStorm
* PhpStorm
* Rider
* CLion
* RubyMine
* GoLand

## 功能

支持在以下场景中提供自动完成及悬浮提示：

* HTML 标签名
* HTML 标签属性名
* HTML 标签属性枚举值
* HTML 标签中的事件名
* CSS 中的 `::part()` 选择器的 `part` 属性名
* CSS 中组件内的 CSS 自定义属性名
* CSS 中的全局 CSS 自定义属性名
* CSS 中的全局 class 名
